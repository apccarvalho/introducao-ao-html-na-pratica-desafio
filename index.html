    6. O que é HTML?

    O HyperText Markup Language - Linguagem de Marcação de Hipertexto - é uma linguagem de marcação que permite a criação de websites, sendo possível a inserção de conteúdos, como imagens e vídeos, estabelecendo a conexão e compartilhamento desses conteúdos.

    Estrutura básica do HTML

    Para a construção de páginas HTML, é utilizada a seguinte estrutura obrigatória:

    • html: Esta tag abre e fecha a estrutura da página, tudo que estiver entre elas será uma página HTML.
    • head: As informações aqui inseridas são carregadas antes das informações que são exibidas aos usuários. É aqui que pode ser inserido o nome que aparece na aba do navegador, por exemplo.
    • body: Aqui é inserido as informações que são exibidas ao usuário.

    Falando sobre tags

    As tags são usadas para informar ao navegador a estrutura do site. São as instruções do que e como será feito e enfim exibido.

    As tags são escritas da seguinte forma: seu nome entre o sinal de maior e menor - < > - para abrir a estrutura a ser construída (como um parágrafo, título ou lista) e para fechar, seu nome entre o sinal de maior e menor com uma barra antecedendo o nome - < / >.

    Algumas das tags utilizadas são:

    • < p > - construção de parágrafos;
    • < h1 > ao < h6 > - utilizada para o tamanho da fonte do título ou cabeçalho. Sendo h1 a "maior" e o h6 o "menor";
    • < strong > - utilizada para colocar o texto em negrito;
    • < i >- utilizada para colocar o texto em itálico;
    • < u > - utilizada para sublinhar o texto;
    • < a > - cria a ancoragem para utilização de links dentro do próprio site ou para fora;
    • < ul > - cria uma lista não ordenada (são aquelas que utilizam a bolinha como marcador do tópico);
    • < ol > - cria uma lista ordenada (são enumeradas);
    • < li > - para escrever os tópicos da lista ordenada ou não ordenada;

      • O HTML é padronizado pela comunidade internacional W3C(The World Wide Web Consortium), mais informações estão disponíveis no site da instituição.

      Para a lista completa de tags, você pode consultar na página W3Schools.

    Atributos das tags

    Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, identificar para onde será a ancoragem do link etc...

    Por exemplo, utilizamos a tag < a > para link e o atributo href ="url" para indicar a url da página para onde o link vai.
